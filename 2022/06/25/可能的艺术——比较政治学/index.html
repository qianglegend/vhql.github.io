可能的艺术——比较政治学

全球视野

  1. 政治比较的维度

    1. 政治比较的尺度（如何衡量政治的发展）：一是政体是否民主（民主问责），二是国家能力（征税能力）是否强大。
    2. 比较的视野是一种俯瞰的视野，在不管分析多么具体的问题，都始终会保持一个比例感，始终提醒你，所见到的只是表象，表象未必是真相，思考的越多，就越相信，智慧的本质是对事物比例的公正判断。

  2. 和平的“爆发”：一个黄金时代？

    1. 冷战结束后，和平是主流，和平背景下，激进和对完美乌托邦的迷恋，反而会对社会文明造成过犹不及的影响和危害。不要让最好成为更好的敌人。
    2. 在全球化格局下，国际体系影响国家命运，国家命运影响个体。
    3. 当前全球化格局，自由霸权崛起，欧美内部的市场自由成为全球的市场自由。历史上每一种国际秩序都对应着其中最强大国家的观念和利益，而权利转移到观念和利益不同的国家时，国家秩序也随之发生变化。
    4. 二战后英美签订的《大西洋宪章》精神，构建起当代国际主流价值观，不寻求领土扩张，推动贸易自由，推动民族自决，人民主权。同时，欧美有能力捍卫《大西洋宪章》的落地。

  3. 全球化的经济后果

    1. 全球化是一个连通器，把世界各地连接起来，总的水位上升时，不同地区的水位升高速度不同。
    2. 现在世界的主要矛盾，不是发达国家和发展中国家的矛盾，而是各国的全球主义者站在一起，与四面八方的民族主义者之间的矛盾。
    3. 特朗普认为，中国是“国家资本主义”，美国是“自由资本主义”，在竞争中不公平。他希望各国都是自由资本主义，但在“有限政府”的限制下，能动用的政策工具只有关税，

  4. 全球化的三难困境

    1. 全球化，主权国家，民主政治难以兼得。中国是兼顾全球化和主权国家，放弃民主政治。
    2. 资本主义的特点是创造性毁灭，但是，当它带来生产率的提高，帮助无数人脱离贫困，激发人的创造性和奋斗精神，这代表了更深的一种人道主义精神。

  5. 全球化和文化反弹

    1. 有文化传播就会有文化反弹，也是“防御性文化民族主义”。人不仅仅是理性的动物，也是认同的动物，他者越有可能吞没“我们”，就越需要强化甚至想象集体“我们”来抵御。曾经越伟大的文明，在面临外来冲击时，反弹越强烈。
    2. 移民速度和文化距离成为文化融合的难点。一个族群越是大规模的，快速的进入他国，往往越没有必要融入周边的环境。

政治的转型

  1. 全球民主化历程

    1. 第一波民主化：19世纪到一战，主要是西方欧美国家。紧接着民主衰退，魏玛共和国的崩溃，日本法西斯崛起，西班牙内战等。
    2. 第二波民主化：二战后西方的卫星国，如南欧，拉美诸国。紧接着民主衰退，大量发展中国家出现政变，内战和革命。
    3. 第三波民主化：20世纪70年代中期至今，由1974年葡萄牙政变开始。

  2. 民主衰退的表现之一：民主崩溃，民主质量的劣化

    1. 民主化的本质是从权利垄断转为权利竞争。这意味着权利不再是代代相传的私有财产，而是成为大家凭能力获得的。在争夺过程中，很容易发生争斗，互相指责作弊，结果爆发冲突。要避免这种情况，需要透明公正的规则，可信的法治，需要各方势力的信任和妥协，这些机制的培育都需要时间，在这个过程中，只要出现一方破坏游戏规则，就会引发旋涡状的恶性循环，劣币驱逐良币，体制被打破。
    2. 威权体制下的权力分配往往是一次性的，一朝得势，长期得势。一次性的游戏中，容易形成你死我活的规则。
    3. 很多转型国家的政治力量，把可重复性的民主化政治，玩成一次性游戏。
    4. 民主的伪劣化，民主质量不佳，表现在具有周期性的选举性质，但是言论自由，新闻自由，结社自由严重倒退。以致其选举是极不公平的政治竞争。

  3. 从民主浪漫主义到民主悲观主义

    1. 越是极端的乐观主义者，越容易转化为极端的悲观主义者。最强烈的乐观容易陷入最深的失望，最深的失望通向最强烈的悲观。
    2. 民主转型在历史上任何国家都遭遇了巨大的冲突，从未一帆风顺。
    3. 在成果上，第三波民主化浪潮呈现相当程度的“民主沉淀”现象。同时，政变现象在世界范围内减少。很多国家开始一边政变，一边承诺将民主选举引入。
    4. 相对于民主衰退，现在应该是民主拉锯，转型从扩展期进入平台期，从吸收期进入消化期。

  4. 民主韧性的来源

    1. 基于当代世界各国人民在观念上更加认同民主。
    2. 但与此同时，世界各地的绝大多数民众并没有把“程序性民主”和“绩效性民主”对立起来，而是都想要。
    3. 人心之变，是理解制度变迁的一把钥匙，也是理解制度沉淀的入口。当政体缺乏观念的支撑，它就像空中楼阁，容易倒塌。
    4. 在现代社会，经济发展让权利不再奢侈，技术让观念传播更加高效，城市化让社会动员成本更低，而中产阶级往往成为政治动荡的缓冲器。这些因素加持下，人们相信社会道义和社会自主性相联系，民主话语权越来越成为政权合法性的“缺省设置”。

  5. 不自由的民主（法国）

    1. 在新兴民主中有一个奇特的现象，一些明明是通过民主选举上台的政治家，却挣脱权利的制约机制，打压反对派的言论和行动自由，以实现权利的巩固。民主在崛起，宪政和自由主义却没有与之同步。
    2. 卢梭构建社会契约论时，敏锐的发现，人民主权观念最大的挑战是政治的派系化。如何防止民主堕落为不同政治势力追逐私利的竞技场，是各个民主国家共同的难题。美国通过以野心对抗野心，以派系制衡派系。
    3. 法国大革命时期的民主构建是上下双向过程，统治者需要打压异己维持自己的统治，民众渴望万众一心短平快的救世方案，因此，不自由的民主主义本质是“威权-民粹”主义。
    4. 在一些新兴国家，民主上升的同时，自由成为牺牲品，一个运动型的政党在一个魅力型的领袖领导下，通过民意赢得权利，再通过权利打压自由赢得更多选票，下次胜利后更加严厉的打压政治自由，由此形成一轮又一轮的恶性循环。民主反而为权利的滥用提供了合法性。
    5. 民主和自由有交叉，但并非一个东西。民主是关于如何产生执政者的规则，自由则是关于限制执政者的规则。学习民主比学习自由容易的多，前者是一种制度，而后者是一种习俗。这也是很多新兴民主国家掉入“不自由民主”陷进的原因。
    6. 人类政治文明的变迁没有捷径，必须穿过千千万万人的心灵。

  6. 从“阿拉伯之春”到“阿拉伯之冬”

    1. 威权倒台≠民主转型
    2. 一个威权政府的倒台更普遍和常见的后果是另一个威权体制的建立，甚至是无政府状态和战乱的出现。

  7. 民主转型常见困境——裂痕动员

    1. 民众并非万众一心，反而是多元的，甚至是两极分化的，在政治动员过程中，这个多元性会变得更清晰和强化，将社会撕裂到水火不容的程度时，民主就走向崩溃。
    2. 在多元化社会结构的“诸善之争”，是自由社会的最大困境。

  8. 民主政体自我颠覆的潜质（南非）

    1. 人们珍视团结的同时也重视自由，自由包含着多元性，只要人们可以自由的运用理智，就会形成不同的意见。并不存在作为整体的人民，只有形形色色的群体。
    2. 对很多国家，推翻威权体制只需要推翻一个统治集团，而克服社会撕裂则需要所有阵营保持克制。
    3. 从南非转型得出的结论：共同的底线。民主转型并不需要全社会就重大问题达成共识，不需要万众一心，但是它需要人们就“如何对待分歧”达成共识。
    4. 共同的底线是什么：胜利者保持宽容，失败者保持耐心。
    5. 很多失败的民主转型，是由于革命斗士太多，太具有战斗性，无法将革命政治转化为常态政治，手里举着锤子，看什么都像钉子。
    6. 自由从不将于人类，人类必须上升其高度。

  9. 民主的功能

    1. 民主通过给予民众制度化的发言权，来解决统治者任意妄为的问题。并非万能药，它取决于人们给它输入什么。

  10. 表亲的专制（印度）

    1. 如果说政府的专制是一种自上而下的政治关系，表亲的专制则是水平方向的社会压迫，其根源来自社会习俗。而社会压迫往往比政府压迫更难以解决。
    2. 当制度的发展超前于文化，文化必然会以暗度陈仓的方式把规则悄悄扭曲为潜规则。
    3. 表亲的专制加剧了政治冲突，抵消了选举意义。在表亲专制下，认同逻辑碾压理性逻辑，只要是我们的人我就支持，不唯才是举，而是唯亲是举。
    4. 市场化下，表亲专制慢慢消解。资本主义虽然在财富方面制造不平等，但是在消除社会等级方面，恰恰是一种平等化力量，它根据你为市场提供的价值，而非种姓，宗教决定你的位置。

  11. 威权浪漫主义

    1. 威权浪漫主义包含人们对政治强人的两种想象，一种是对其道德智识水平的想象，一种是对其统治能力的想象。
    2. 威权体制包含极大的不确定性，在民主体制下，如果社会治理糟糕，常常是因为人们选错了领导人，通过选举可以修正。但是威权体制下，虽然也有适应性学习，但是这种纠错不是制度化的，往往需要依赖某个领导人的去世，某场政变，某次战争或者某个开明的领导人胸怀和智识，才能实现转变。

  12. 民主的价值与其时间的厚度相关。

国家的建构

  1. 国家建构特制政治发展的一个维度，即国家能力的发展。

  2. 国家是特定疆域内合法垄断暴力的机构，这是与其它组织的根本区别——马克思·韦伯

  3. 正常情况下，暴力的分散化才是自然状态，暴力的垄断化，是脱离了自然引力的人为状态。

  4. 暴力从分散走向垄断，面临两大难题：

    1. 权利的集中化，谁是老大；是暴力垄断如何获得内部承认。
    2. 领土范围的清晰化；是暴力垄断如何获得外部承认。

  5. 亨廷顿认为：国家之间最重要的政治差异不是政府的形式，而是政府的深度。民主和专制的差异。小于体现着组织性，有效性的国家和缺乏这些特质国家间的差异。

    以此推论，东亚国家之所以可以从所有发展中国家率先摆脱贫困，是因为有着强大而积极的官僚体系。

  6. 为什么许多新兴国家和现存的民主国家，民主表现的令人失望，因为国家的能力没有跟上人们对民主的需求。一个强力的国家，一定是实现了暴力高度垄断的国家。暴力的集中化将会使得暴力减少。

  7. 国家的另一项功能，提供公共服务，则需要依赖高效的征税能力。

  8. 在其他方面相当的情况下，国家建构比较成熟的地方，其治理表现通常更优越。将会具有极长的历史惯性。

  9. 国家能力来源自战争，弱肉强食的时代下，战争强化那些幸存国家的国家能力。

    战争缔造国家，国家制造战争。战争的密度和烈度的提升，使得只有强有力的国家才能赢得战争。这也促使常备军的建设，中央财税机制的形成，官僚行政体系的建立和扩张。

    在封建体制转向中央集权制的过程中，权利被集中，意味着国家能力的增强。

  10. 以秦国为例，商鞅变法破诸侯，立中央，解构社会，强化政府，打破社会中无数的小共同体，打造“万众一心”的大共同体。

    同样为什么欧洲没有形成大一统王国，作者认为，从16世纪开始国家建构的时候，欧洲的中央权利已经深深嵌入一个相互制衡的权利网络之中，除了王权，还有强大的教权，高度自治的城市，崛起的商贸阶层，以及势力更加强大的贵族权利，这些势力约束了“国家建构”的速度。

  11. 国家构建后，通过文官制，稳固国家能力。

    1. 以中国为例，通过政府条条块块编织行政网络。官僚制体系的创新是去除封建诸侯，立中央集权。也就是皇权之下，官职是皇帝随时可以收回的帽子，官职就成为了可再生资源，在每一次封官时，都重新树立了自己的权威。维持大一统的力度，和官僚制强大的向心力关系巨大。

    2. 同时，文人当政，因为文人没有军事基础，经济基础，宗教基础，恰恰可以成为王权最安全的朋友。

    3. 此外，科举制的出现，让底层人们有了上升通道，避免了培养出皇权之外的独立力量，将四处冲撞的精英冲动，纳入到统一的轨道上，通过读书做官，为皇帝效力。

    4. 文官制大大强化了古中国的国家能力，并使之成为一种深厚的传统，王朝可以灭亡，但是政治文明生生不息。但同样，过于强大的官僚体系，也意味着国家淹没社会，取消社会，官与民的不对称力量，也使得民主思想启蒙受限。

  12. 美国的国家建构，从一开始，美国人建立的就是一个“反国家”的国家，它把中央集权视为洪水猛兽进行堤防，在国家结构上采取“平行式权利”，联邦政府和州政府都有自己的权利领地，彼此平行而不是上下级。

    政治上，美国是强地方，弱中央。经济上，是强市场，弱政府。

    在近一百年，美国国家建构，政府职能的扩大，联邦权利的提升，其动力来自民众。虽然现在政府力量很强，但是社会力量也很强，政府是一只被关在笼子里的猛虎。这个转变源于下面：

    1. 进步主义时代，信息开放与流通，人性的进步与发展；
    2. 罗斯福新政，罗斯福作为进步主义者的胜利；
    3. 民权运动时代；

    从美国的国家建构分析，其核心动力来自社会运动，社会运动需要制度空间，以自由为前提，以民主为传导机制，其本质是政治权利的力量。通过社会压力给国家赋权，是“另类道路”的启示。

    当一个国家能力发展远远超过社会能力，它往往不会再有动力来给民众赋权。更重要的是，当国家接纳民权，其国家能力未必受损，反而会因合法性的上升而国家能力上升。

    美国经验的真正特色——一个强大的政府，被一个更强大的社会和市场环绕。国强不必民弱，甚至民强国可以更强，这一点鼓舞人心。

  13. 国家建构的失败因素（阿富汗为例）

    1. 多山地形——中央集权政府难以形成，造成部落式传统；
    2. 地缘位置——内战往往成为大国代理战争，难以胜负，暴力垄断格局难以形成；
    3. 宗教极端主义——世俗力量难以建立；
    4. 民族结构——各方政治势力权利分配难以达成共识；

    极权追求乌托邦的风险，乌托邦过于乌托邦，所以要实现只能诉诸于极端手段，而极端助长极端，最终在相互厮杀中完成对彼此的哺育。

政治文化

  1. 这个世界的老大是谁，将会深深影响历史前进的方向，因为，老大会试图在全世界复制自己的形象。

  2. 经由观念变迁推动的制度变迁才是最牢靠，坚固的。观念一旦形成，往往具有相当的韧性。全民的抗争会无限提高旧制度的运转成本，但是，真正给旧制度带来致命一击的，是体制内人开始动摇

  3. 观念的来源——本质是经济发展的结果，吃饱穿暖后，人们开始关心权利和自由等价值了。

    启蒙思想的致命吸引力，不是因为它来自西方，而恰恰是因为它来自我们自己的内心。所谓启蒙，未必是把一个外来的什么思想塞到我们的脑子里，而是用一盏灯把本来就在我们心灵深处的东西给照亮了而已。

    源于道德直觉的正义感有种令人敬畏的天真，你会发现，当所有政治的泥沙沉淀，所有的理论波涛平息，所有流行的趋势过去，最终，这种无与伦比的天真还是会从水底浮现，会诱惑你向他伸手去。

  4. 哪里有反抗的机会，哪里才会有反抗。

  5. 一个国家的社会资本丰富与否，决定它的民主是否能走向稳固和良性运转。社会资本指社会自发结社的习性。

    在一个社会30%人口强调自我表达的价值时，形式上的民主倾向于出现，而当一个社会45%的人口强调自我表达的价值时，实质性的民主倾向于出现。
